# 機能

- Notionの見た目をWebアプリで再現する
- フィルター付きの認証（DiscordのHorizonサーバー参加者のみログイン可）
- 一日一回の自動更新
    - 毎日am１０時にNotionの内容を読み取ってアプリ内容を更新します
- タグ検索
- 進捗確認
- 解答表示
    - 進捗データを読み取り、回答済みの問題の解答のみ表示されます
- ロールによる閲覧制限
    - 認証時にDiscordからロールを取得。ロールによってみられるカリキュラムを制限します
- Notionでのページ制御
    - 公開非公開
    - 更新非更新
        - 基本的には編集されたNotionページを読み取ってAtlasを更新しますが、更新したくないページは更新しない設定にNotionでできる
    - ページのタグつけ
    - ページのカテゴリー
    - ロールによる閲覧制限
    - ページの表示順
