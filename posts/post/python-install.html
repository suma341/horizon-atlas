Pythonインストール方法

1. ダウンロード

まず、Python公式サイト（https://www.python.org/downloads/）にアクセスしてください。

「Download Python 3.12.2」をクリックして、インストールファイルをダウンロードしましょう。

2. インストール

ダウンロードが完了したら、インストールをしましょう。ブラウザの右上のアイコンからインストールファイルを実行するか、エクスプローラーから実行しましょう。

プログラムの作成・実行

引き続き、Pythonのプログラムを作成して実行する方法を紹介します。

まず、「IDLE (Python 3.12 64-bit)」というアプリを探し、実行しましょう。

すると、このようなウィンドウが立ち上がります。

左上の「File」をクリックし、「New File」を選択することで新しいプログラムを作成できます。もうひとつ、このようなウィンドウが立ち上がります。

これが、実際にPythonのプログラムを書いていくエディタという画面です。

なにかプログラムを書いてみましょう。例えば、print("Hello World!")っと…

プログラムを書いたら、必ず保存しましょう。左上の「File」をクリックして、「Save as…」を選択すると、ファイルを保存する場所と保存するファイルの名前を選べるようになります。

同様のことが、キーボードの Ctrlキーと Sキーを同時押しすることでも可能です。

保存ができたらプログラムを実行してみましょう。ウィンドウ上部の「Run」をクリックし、続いて「Run Module」を選択します。

最初に起動したウィンドウに「Hello World!」と表示されれば成功です。

Run → Run Module を選択せずとも、キーボードの F5キーを押すだけで同じように実行ができます。一度試してみてください。

これから行うプログラミングは、この 書く→保存する→実行する の繰り返しです。慣れないうちは操作に手間取るかもしれませんが、先輩たちがサポートしてくれるので、少しづつできるようになりましょう！
